今天我將分享如何使用 Tailwind CSS 自訂我的 Astro 部落格主題。Astro 和 Tailwind 的結合使得樣式設計變得非常高效。

## 關鍵自訂

1. **顏色方案**: 創建了一個自訂的暗/亮模式主題
2. **排版**: 設置了一致的字體比例
3. **佈局**: 實現了響應式設計模式
4. **元件**: 樣式化可重用的 UI 元件

## 主題開發提示

- 從清晰的設計系統開始
- 使用 Tailwind 的設定檔進行自訂值
- 利用 CSS 變數進行動態主題
- 在不同裝置和螢幕尺寸上進行測試
